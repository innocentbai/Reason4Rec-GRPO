基于大语言模型的推理式推荐系统复现与 GRPO 优化
---
• 采用三阶段协同推理架构，构建Summarizer→Reasoner→Predictor的渐进式推理链路。
通过偏好提取、推理生成与评分预测的协同工作，实现从历史行为到推荐决策的可解释推理过程。
• Reason4Rec 引导模型在预测前生成用户偏好分析和推荐理由， 提升推荐准确性与可解释性。
• 采用知识蒸馏+ 强化学习的混合训练范式，首先利用GPT 生成多样化候选理由，通过专门训练的奖励模型筛选出评分预测误差最小的高质量理由作为Reasoner 监督信号，指导学习生成可信且与评分一致的推荐解释。最后使用GRPO 算法对推理质量进行强化优化。
• MAE 从0.5585 降低至0.5450，RMSE 从0.7855 降低至0.7637。
